# Security Audit

Perform comprehensive security audit checking for vulnerabilities and security best practices.

Invoke qa-agent in security analysis mode to audit the code. The agent will check for OWASP Top 10 vulnerabilities including injection flaws, broken authentication, sensitive data exposure, XSS, and provide a detailed security report.
