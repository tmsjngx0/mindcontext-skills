# Analyze Code

Perform comprehensive code analysis to find bugs, security issues, edge cases, and potential problems.

Invoke qa-agent in bug-hunting mode to analyze the code. The agent will check for common vulnerabilities, logic errors, edge cases, and provide a detailed report of all issues found.
